Meet App

I build this procejt with the aim to find tech event in a city near you! It was written in React JS using a TDD and BDD approach, I also used a serverless function for oAuth2 with google and Google calender Api to fetch the events

Technologies used to build this app includes: Serverless functions, oAuth2 with Google, Google Calender Api, React, OOP
